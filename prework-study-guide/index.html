HTML

  • HTML - HyperText Markup Language
  • The head element contains information about the webpage.
  • The body element represent the visible content shown to the user.
  • A block-level element always starts on a new line and takes up the full width available.
  • An inline element does not start on a new line and it only takes up as much width as necessary
  • A semantic element clearly describes its meaning to both the browser and the developer.
  • The a tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to.

CSS

  • If you set display to flex, all children inside of the container become a flex item. They will be automatically arranged into a row and each element will be the same width as their content. The flex container spans its own width, and each child has to fit in the container's width.
  • If you set display to inline-flex, it makes the flex container behave like an inline element.
  • flex-direction can change the direction from horizontal or vertical with 'row' & 'column'.

Git

  • git status: check what branch we are currently working on
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add: add a file as it looks now to the next commit.
  • git commit -m "message": commit staged content as a new commit snapshot.
  • The git push command is used to upload local repository content to a remote repository. Pushing is how you transfer commits from your local repository to a remote repo.

JavaScript

  • Control flow is the order in which a computer executes code in a script.
  • When we create a function, we are defining a block of code that performs a specific task when called.
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
